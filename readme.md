#### 快捷键

1. 切换 mac 左右屏： control + fn + 左右键
2. 跳转文件顶部、底部、行首、行尾：command + fn + 上下左右
3. 切换文件 tab 的当前页：ctrl + 12345
4. 侧边栏显/隐：Ctrl+B
5. Ctrl + ~
6. option + command + 左右
7. 折叠和展开代码：option + command + {和}
